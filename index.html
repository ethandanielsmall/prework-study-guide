HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The "< meta >" tag is short for metadata.
  • Metadata is the info on a webpage that is not seen by the user.

CSS

    • A margin indicates how much space we want around the outside of an element.
    • A padding indicates how much space we want around the content inside an element.
    • The difference between HTML and CSS is that CSS is the style of the page, and HTML is the content.
    • CSS is an easier way to style a page. HTML can make a page, but it will not style as easily.
    • rel = the attribute specifies the relationship between the current document and the linked source.
    • href = the attribute specifies the location of an external source.

Git

  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • Lesson 2.4.6 in the prework has a walkthrough of how to commit, push, pull, etc, if you get stuck in github.
  • Github stores information in 'the cloud', aka a server other than a personal computer.
